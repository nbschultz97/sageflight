Why Sageflight

Your aircraft is giving you signals. Sageflight translates.

A flag, a noise peak, or a stale config should end in a decision—not an evening of searching. Sageflight turns raw bench evidence into a clear cause and a safe next move.

Live signal · arming flags

"Why won't it arm?"

The #1 beginner wall. The arming doctor decodes every arming-disable flag live and tells you the fix — not just the flag name. The AI can read them too.

Flight evidence · blackbox

Turn noise into a tuning decision

The Blackbox tab reads the full tuning state from your log, decodes flight frames, charts gyro noise spectra and motor stats — and the AI reviews it with concrete set suggestions citing measured numbers.

Version change · guarded write

Migrate without gambling on old settings

Pasting an old diff into new firmware silently drops renamed parameters — the classic burned-motor trap. The AI translates your diff for the target version; you review every line before it runs.

Safety Model

The LLM never touches the aircraft

An AI that can spin motors and flash firmware needs more than a disclaimer. Sageflight layers safeguards the LLM cannot bypass — enforced in the API, not the prompt.

Human confirmation tokens

Every motor actuation, ESC interrogation, config write, and firmware flash requires a per-action token issued only after explicit checkbox confirmation (props off, restrained, backup taken). Tokens are single-use and expire in 60 seconds.

Hard caps in the API

PWM clamped to 1000–1300, ≤5 s per spin, motors always commanded back to 1000 in a finally block. CLI writes re-validated server-side. Flashing refuses to run without a backup or with multiple DFU devices attached. All serial access serialized through a mutex.

No direct actuation path

The AI's tools are read-only except propose_config_changes — which only drafts commands. Server-side validation, the backup requirement, and your explicit Apply click sit between any proposal and the flight controller.

Hallucination policy: assume it, contain it, shrink it. A wrong sentence is possible. A wrong actuation is not.

$ ollama pull nomic-embed-text   # embeddings for the local docs index
